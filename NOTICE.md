# Notice

**yoyo Illustrations** is an adaptation of the **Ian Xiaohei Illustrations** AI-agent skill created by Ian.

The illustration methodology, style DNA, composition patterns, and workflow originate from Ian's `ian-xiaohei-illustrations` skill. This repository adapts that skill by:

- replacing the recurring character **小黑 (xiaohei)** with **yoyo**, a soft purple octopus;
- making the handwritten annotations multilingual (written in the source article's language);
- making the skill runtime-portable (Claude Code, Codex/OpenAI, yoyo, and other agent tools), with no hard dependency on any single runtime's image-generation tool.

Please keep attribution to Ian for the original skill when redistributing or further adapting this work.
